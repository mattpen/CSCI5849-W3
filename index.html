Accessibility Features

  • Alt text is provided for images used by default, and requested for user input images.
  • Keyboard navigation (standard TAB, Enter, Space nav) works throughout the entire app.
  • ARIA-live regions are employed to manage updates for the tab navigation and Contact List. ARIA roles are defined to help screen readers distinguish between the navigation and main content.
  • Custom Keyboard Shortcuts are provided for the TAB navigation and for the Dial tab. The following keys may be used in combination with the Alt key in order to bypass the standard TAB navigation.
    Key Function
    r Navigates to the Dial tab.
    l Navigates to the Contact List tab.
    a Navigates to the Add Contact tab.
    h Navigates to the Help tab.
    0-9, *, # Enters the given character in the dialer input on the Dial tab.
    c Initiates a call using the input on the Dial tab.
    x Clears the input field on the Dial tab.
